# German Credit Risk Analysis

##  Project Overview

This project analyzes customer credit data to identify patterns influencing credit risk classification.  
The analysis focuses on demographic attributes, financial indicators, and loan characteristics.

Dataset used: German Credit Dataset

---

##  Objectives

- Perform structured data cleaning in Excel
- Conduct exploratory data analysis (EDA)
- Identify risk-driving factors
- Build visual analytics in Tableau
- Demonstrate analyst-level project workflow

---

##  Data Preparation

Data cleaning and validation performed using Excel:

- Missing value handling
- Category normalization
- Numeric validation
- Feature engineering
- Banding & segmentation

---

##  Key Analysis Performed

- Credit Risk Distribution
- Risk vs Age Band
- Risk vs Credit Amount
- Risk vs Duration
- Exposure Segmentation
- Financial Behavior Patterns

---

##  Tools & Skills Demonstrated

- Excel (Data Cleaning, Pivot Tables, Feature Engineering)
- Tableau (Visual Analytics & Dashboarding)
- Data Analysis & Interpretation
- Business Insight Extraction
